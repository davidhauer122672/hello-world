## Deployment Instructions

### Step 1: Platform Selection

| Platform | Best For | Notes |
|----------|----------|-------|
| Banana Pro AI | Builds 1 and 2 (static live wallpapers) | Strongest photorealistic video generation |
| Grok Mobile App | Build 3 (live talking avatars) | Native conversational avatar support |
| intoLive App | Converting .mov to Live Photo | Required for iPhone wallpaper activation |

### Step 2: Generation Workflow

1. Open Banana Pro AI (or Grok for Build 3).
2. Copy the entire prompt block for the target build.
3. Paste directly into the generation input field.
4. Set output format to .mov if the platform offers format selection.
5. Generate. Review output against the technical standards table above.
6. If quality falls short on any parameter, append this refinement note to the prompt: "Increase subsurface scattering intensity by 20%. Add more visible pore detail on nose and cheeks. Ensure eye moisture film is visible in close-up."

### Step 3: iPhone 16 Pro Deployment

1. Transfer the .mov file to iPhone via AirDrop or Files app.
2. Open the intoLive app (free, App Store).
3. Import the .mov file and convert to Live Photo.
4. Go to Settings > Wallpaper > Add New Wallpaper.
5. Select the Live Photo from your library.
6. Enable "Live" toggle to activate dynamic motion on lock screen.
7. Long-press the lock screen to see the animation play.

### Step 4: Grok Avatar Integration (Build 3 Only)

1. Open Grok app on iPhone 16 Pro.
2. Navigate to Companion Mode or Avatar settings.
3. Upload the generated character assets.
4. Assign voice profiles: Character 1 (bright Australian), Character 2 (sultry, husky).
5. Set role context to "Coastal Key Enterprise Executive Administrator."
6. Test conversational flow with a sample CEO briefing prompt.
