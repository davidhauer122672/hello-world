## Location Strategy

### Target Parameters

| Parameter | Requirement |
|-----------|-------------|
| Market | Stuart / Jensen Beach / Port St. Lucie corridor |
| Size | 1,200 sq ft (+/- 10%) |
| Type | Retail/office flex space, ground floor preferred |
| Visibility | High-traffic road or shopping center |
| Parking | Minimum 4 dedicated spaces |
| Lease Term | 3-year initial with 2x 3-year renewal options |
| Budget | $18-$24/sq ft/year NNN ($1,800-$2,400/month base) |
| Move-In | Target Q3 2026 (post-break-even) |

### Site Selection Criteria (Weighted)

| Criterion | Weight | Notes |
|-----------|--------|-------|
| Proximity to target demographics | 30% | Near seasonal/luxury residential areas |
| Visibility & signage potential | 25% | Road frontage, monument signage |
| Lease economics | 20% | NNN, CAM caps, TI allowance |
| Parking & accessibility | 15% | ADA compliant, easy access |
| Build-out requirements | 10% | Minimal renovation needed |

## Financial Model

### Startup Costs (One-Time)

| Item | Cost | Notes |
|------|------|-------|
| Security deposit (2 months) | $4,800 | Refundable |
| Build-out & renovation | $8,000 | Paint, flooring, basic walls |
| Furniture & fixtures | $5,000 | Reception, desks, conference table |
| Technology infrastructure | $3,500 | Network, displays, workstations |
| Signage (exterior + interior) | $2,500 | Monument + window + lobby |
| Command Center display (65") | $1,200 | Live fleet dashboard |
| Coffee/beverage station | $500 | Client experience |
| Branding & decor | $1,500 | Wall wraps, brochures, brand elements |
| **Total Startup** | **$27,000** | |

### Monthly Operating Costs

| Item | Monthly | Annual | Notes |
|------|---------|--------|-------|
| Base rent (NNN) | $2,100 | $25,200 | $21/sf/year midpoint |
| CAM/NNN estimates | $400 | $4,800 | Insurance, taxes, maintenance |
| Utilities | $250 | $3,000 | Electric, water, internet |
| Internet (business fiber) | $150 | $1,800 | Required for operations |
| Insurance (GL + BOP) | $200 | $2,400 | Business liability |
| Cleaning service | $150 | $1,800 | Weekly |
| Office supplies | $75 | $900 | Minimal |
| Coffee/beverages | $50 | $600 | Client hospitality |
| **Total Monthly** | **$3,375** | **$40,500** | |

### Revenue Attribution Model

The retail location drives revenue through three channels:

| Channel | Monthly Contribution | Margin |
|---------|---------------------|--------|
| Walk-in conversions (2-3/month) | $600-$900 | 80%+ |
| Client experience upgrades | $200-$400 | 90%+ |
| Brand credibility lift (CAC reduction) | $500-$1,000 | Indirect |
| **Estimated Monthly Contribution** | **$1,300-$2,300** | |

### Break-Even Analysis

| Scenario | Monthly Fixed | Required Revenue | Properties Needed |
|----------|-------------- |-----------------|------------------|
| Conservative | $3,375 | $3,375 | ~15 (at $225 avg) |
| Moderate | $3,375 | $4,500 (30% margin) | ~20 |
| Aggressive | $3,375 | $6,750 (50% margin) | ~30 |

**Trigger**: Open retail location when portfolio reaches 25+ properties (de-risked).

| Margin | 65% |

### Tier 2: Home Watch Premium
| Parameter | Value |
|-----------|-------|
| Price | $249/month |
| Inspections | Weekly (52/year) |
| Includes | Everything in Essential + vendor coordination, storm protocol, concierge |
| Target | Luxury homeowners, frequent travelers |
| Margin | 60% |

| Parameter | Value |

| Price | $399/month or 8% of rental income |

## Client Experience Standards

### In-Office Experience
- Greeted within 30 seconds of entry
- Command Center dashboard visible from lobby (fleet operations, real-time status)
- Branded coffee/water offered immediately
- Consultation room available within 5 minutes
- Proposal generated on-site within 15 minutes (AI-powered)
- Contract signed digitally same-visit (target 40% close rate)
