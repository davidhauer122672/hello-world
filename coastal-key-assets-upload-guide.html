2. R2 Bucket Creation Steps

  • Navigate to dash.cloudflare.com
  • Click R2 Object Storage in left sidebar
  • Click Create Bucket
  • Name: coastal-key-assets | Location: Automatic
  • Go to Settings tab → Public Access → Allow Access
  • Copy the Public Bucket URL
  • Create R2 API Token (Object Read & Write)
  • Save Access Key ID and Secret Access Key

3. Zapier Zap Architecture

TRIGGER Airtable / Form / Webhook → ACTION 1 Upload to R2 (S3 API) → ACTION 2 Log to Airtable

8. Verification Checklist

  • R2 bucket coastal-key-assets created
  • Public access enabled and URL copied
  • API token generated with Read/Write scope
  • Zapier trigger configured and tested
  • S3 upload action connects to R2 endpoint
  • File uploads to correct path in bucket
  • Public URL returns uploaded file
  • Airtable record created with all fields
  • Error notifications configured in Zapier
