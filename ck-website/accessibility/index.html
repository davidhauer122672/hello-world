Accessibility statement.

Our commitment

Coastal Key Property Management LLC is committed to ensuring that our website is accessible to all visitors, including people with disabilities. We strive to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 at Level AA. These guidelines explain how to make web content more accessible to people with a wide range of disabilities.

Conformance status

We believe that coastalkey-pm.com substantially conforms to WCAG 2.2 Level AA. We continually review and improve our site to maintain and exceed this level of accessibility.

Accessibility features

Our website includes the following accessibility features:

  • Skip navigation: A "Skip to main content" link is provided at the top of every page, allowing keyboard users to bypass repetitive navigation and jump directly to the page content.
  • Semantic HTML: We use proper HTML5 semantic elements (header, nav, main, section, article, footer) to convey document structure to assistive technologies.
  • Keyboard navigation: All interactive elements, including links, buttons, form fields, and navigation menus, are fully operable using a keyboard alone.
  • Focus indicators: Visible focus indicators are provided for all interactive elements. Focused elements display a 2px gold ring so keyboard users can always identify which element has focus.
  • Alternative text: All meaningful images include descriptive alternative text. Decorative images are marked appropriately so they are ignored by screen readers.
  • Form labels: All form inputs are associated with descriptive labels using the label element, ensuring screen readers announce the purpose of each field.
  • ARIA landmarks: ARIA landmark roles (banner, navigation, main, contentinfo) are used to help assistive technology users navigate the page structure efficiently.
  • Color contrast: Text and interactive elements meet or exceed the WCAG 2.2 Level AA minimum contrast ratio of 4.5:1 for normal text and 3:1 for large text.
  • Reduced motion support: We respect the prefers-reduced-motion media query. When a visitor has indicated a preference for reduced motion in their operating system settings, animations and transitions are minimized or disabled.

Known limitations

While we strive for full accessibility, some content may not yet be fully accessible. We are actively working to identify and resolve any remaining barriers. If you encounter an issue, please let us know.
